利用jquery动态插入的方式在wordpress网站的菜单栏那里添加社交媒体和语言切换栏

Posted on 2018-08-09 Edited on 2020-10-11

笔者曾写过一篇充分运用jquery动态插入的方式添加微信icon到wordpress的顶部的方案，作为延伸，笔者继续发挥jquery在这方面独特的优势：利用jquery动态插入的方式在wordpress网站的菜单栏那里添加社交媒体和语言切换栏。

首先我们引入jquery这个库：

<script src="https://cdn.bootcss.com/jquery/3.3.1/jquery.min.js"></script>

为了能够减少对第三方资源的请求，或者减少跨域请求以加快加载速度，笔者把这个放在主题的js文件下，然后引用：

<script src="<?php bloginfo('template_url'); ?>/js/jquery.min.js"></script>

由于这个社交媒体icon主题中引用不灵活，笔者还引入了一个font awesome这个字体图标库。于是下载了这个库的资源。

然后在wordpress头部引用进来，

<link href="<?php bloginfo('template_url'); ?>/css/font-awesome.min.css" rel="stylesheet">

字体包里面有个fonts文件夹，这个需要放在主题目录下，有的话就把里面的内容上传，没有的话就新建一个fonts文件夹，然后把里面的字体包上传即可。

我们打开这个font-awesome.min.css不难发现

@font-face {
  font-family: 'FontAwesome';
  src: url('../fonts/fontawesome-webfont.eot?v=4.7.0');
  src: url('../fonts/fontawesome-webfont.eot?#iefix&v=4.7.0') format('embedded-opentype'), url('../fonts/fontawesome-webfont.woff2?v=4.7.0') format('woff2'), url('../fonts/fontawesome-webfont.woff?v=4.7.0') format('woff'), url('../fonts/fontawesome-webfont.ttf?v=4.7.0') format('truetype'), url('../fonts/fontawesome-webfont.svg?v=4.7.0#fontawesomeregular') format('svg');
  font-weight: normal;
  font-style: normal;
}

它这里是引用了字体包里的内容的，所以，我们把font-awesome.min.css放入css文件夹（与fonts文件夹同级）里面。

好的，这里字体图标已经上传好了，那么我们就把js写入footer.php里面，首先我们解决一下冲突先：

var jQuery_3_3_1 = $.noConflict(true);

因为，笔者刚才引入的是这个版本。经过笔者实践发现其实可以不用根据版本来命名这个变量的，写成jQuery照样没问题。

好的，接下来会有点小技巧在里面，仔细看：

我们想要在菜单栏目那里添加社交媒体icon，那么我们可以先添加一个菜单项，目的是获取这个菜单项的html代码，笔者这里添加通过开发者工具获取到了添加的这个菜单项的代码：

这里可以看出我们把获取右边红色框中的html代码就是社交媒体的其中一个，这样理解是没有问题的。

<li id="menu-item-656" class="menu-item menu-item-type-post_type menu-item-object-page menu-item-home menu-item-top-level menu-item-top-level-1"><a href="#" itemprop="url"><span class="avia-bullet"></span><span class="avia-menu-text">Home</span><span class="avia-menu-fx"><span class="avia-arrow-wrap"><span class="avia-arrow"></span></span></span></a></li>

好的，我们把多余的代码去除掉，最后简化成如下：

<li id="menu-item-656" class="menu-item menu-item-type-post_type menu-item-object-page menu-item-home menu-item-top-level menu-item-top-level-1"><a href="#" itemprop="url"><span class="avia-menu-text">Home</span></a></li>

所以我们添加四个社交媒体链接对应的代码如下：

<li id='menu-item-599' class='menu-item menu-item-type-custom menu-item-object-custom menu-item-mega-parent menu-item-top-level menu-item-top-level-1'><a href='#' itemprop='url'><span class='avia-menu-text'><i class='fa fa-facebook'></i></span></a></li>
<li id='menu-item-600' class='menu-item menu-item-type-custom menu-item-object-custom menu-item-mega-parent menu-item-top-level menu-item-top-level-1'><a href='#' itemprop='url'><span class='avia-menu-text'><i class='fa fa-amazon'></i></span></a></li>
<li id='menu-item-601' class='menu-item menu-item-type-custom menu-item-object-custom menu-item-mega-parent menu-item-top-level menu-item-top-level-1'><a href='#' itemprop='url'><span class='avia-menu-text'><i class='fa fa-skype'></i></span></a></li>
<li id='menu-item-602' class='menu-item menu-item-type-custom menu-item-object-custom menu-item-mega-parent menu-item-top-level menu-item-top-level-1'><a href='#' itemprop='url'><span class='avia-menu-text'><i class='fa fa-instagram'></i></span></a></li>

这四个社交媒体链接就算添加完毕了，接下来我们在这四个社交媒体的后面添加语言切换的功能，那么为了确保风格样式一致，那么我们复制一个社交媒体的代码，然后添加语言切换的下拉框代码：

<div class="country">
    <select onchange="window.location=options[(this.selectedIndex)].value">
        <option value="/#en">EN</option>
        <option value="/#cn">CN</option>
        <option value="/#fr">FR</option>
    </select>
</div>

这样添加之后就可以选中其中一个的时候就可以跳转到响应的地址。

<li id='menu-item-603' class='menu-item menu-item-type-custom menu-item-object-custom menu-item-mega-parent menu-item-top-level menu-item-top-level-1'>
    <a href='#' itemprop='url'>
        <span class='avia-menu-text country'>
            <select onchange='window.location=options[(this.selectedIndex)].value'>
                <option value='/#en'>EN</option>
                <option value='/#cn'>CN</option>
                <option value='/#fr'>FR</option>
            </select>
        </span>
    </a>
</li>

可是这个时候，由于我们外面有一个a标签包裹着，当选中的时候，会不发生挑战，因为这个时候，我们需要让a标签失去链接的作用，处理方式如下：

<a href="#" onclick = "return false;" >dd</a>

这里我们就可以成功阻止了。

好的，综合以上的代码，最后添加的js代码如下：

<script type="text/javascript">
    var jQuery_3_3_1 = $.noConflict(true);
    var socialmedia = jQuery_3_3_1("<li id='menu-item-599' class='menu-item menu-item-type-custom menu-item-object-custom menu-item-mega-parent menu-item-top-level menu-item-top-level-1'><a href='#' itemprop='url'><span class='avia-menu-text'><i class='fa fa-facebook'></i></span></a></li><li id='menu-item-600' class='menu-item menu-item-type-custom menu-item-object-custom menu-item-mega-parent menu-item-top-level menu-item-top-level-1'><a href='#' itemprop='url'><span class='avia-menu-text'><i class='fa fa-amazon'></i></span></a></li><li id='menu-item-601' class='menu-item menu-item-type-custom menu-item-object-custom menu-item-mega-parent menu-item-top-level menu-item-top-level-1'><a href='#' itemprop='url'><span class='avia-menu-text'><i class='fa fa-skype'></i></span></a></li><li id='menu-item-602' class='menu-item menu-item-type-custom menu-item-object-custom menu-item-mega-parent menu-item-top-level menu-item-top-level-1'><a href='#' itemprop='url'><span class='avia-menu-text'><i class='fa fa-instagram'></i></span></a></li><li id='menu-item-603' class='menu-item menu-item-type-custom menu-item-object-custom menu-item-mega-parent menu-item-top-level menu-item-top-level-1'><a href='#' onclick='return false;' itemprop='url'><span class='avia-menu-text country'><select onchange='window.location=options[(this.selectedIndex)].value'><option value='/#en'>EN</option><option value='/#cn'>CN</option><option value='/#fr'>FR</option></select></span></a></li>");
       jQuery_3_3_1("#menu-item-search").before(socialmedia);
</script>
