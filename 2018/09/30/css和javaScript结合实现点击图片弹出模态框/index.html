css和javaScript结合实现点击图片弹出模态框和基于HTML5的鼠标悬浮显示tooltip自定义属性的内容

Posted on 2018-09-30 Edited on 2020-10-11 In javascript

不知道多少次有多少客户问我，可否做一个让图片点击弹出一个模态框的，然后在图片上显示这个图片的标题，可想而知这个需求还是很普遍的，由于时间关系一直没有整理出来，这次抽空整理了下，写一个demo也让后面的人随时采用。这里做两个demo：

  1. 针对单张图片实现点击图片弹出模态框
  2. 针对多张批量式的图片显示模态框

针对单张图片实现点击图片弹出模态框

HTML代码

<!DOCTYPE html>
<html>
<head>
 <meta charset="UTF-8">
 <title>close</title>
 <style>
  #real{
  /*点击弹出模态框的图片*/
  margin: 30px;
  max-width: 250px;
  border-radius:6px;
  }
  #real:hover{
  opacity: 0.6;
  }
  #mo{
  display: none;
  width: 100%;
  height: 100%;
  position: fixed;
  overflow: auto;
  background-color: rgba(0,0,0,0.7);
  top: 0px;
  left: 0px;
  z-index: 1;
  }
  #moimg{
  display: block;
  margin:25px auto;
  width: 60%;
  max-width: 750px;
  }
  #caption{
  text-align: center;
  margin: 15px auto;
  width: 60%;
  max-height: 750px;
  font-size: 20px;
  color:#ccc;
  }
  #moimg,#caption{
  -webkit-animation: first 1s;
  -o-animation: first 1s;
  animation: first 1s;
  }
  @keyframes first{
  from{transform: scale(0.1);}
  to{transform: scale(1);}
  }
  .close{
  font-size: 40px;
  font-weight: bold;
  position: absolute;
  top: 20px;
  right: 14%;
  color:#f1f1f1;
  }
  .close:hover,.close:focus{
  color:#bbb;
  cursor:pointer;
  }
  @media only screen and(max-width:750px ) {
  #moimg{
   width: 100%;
  }
  }
</style>
</head>
<body>

<img src="collab4.jpg" id="real" alt="时尚话题">
<!--图片模态框 -->
<div class="motai" id="mo">
 <span class="close" id="close">×</span>
 <img class="motaiimg" id="moimg">
 <div id="caption"></div>
</div>
<script>
  var imgs = document.querySelectorAll("#container img");
  var motai=document.getElementById('mo');
  var moimg=document.getElementById("moimg");
  var realimg=document.getElementById("real");
  var caption=document.getElementById("caption");
  console.log(imgs);
  console.log(imgs.length);
  realimg.onclick=function(){
    motai.style.display="block"
    moimg.src=this.src
    caption.innerHTML=this.alt
  }

   imgs.onclick=function(){
    motai.style.display="block"
    moimg.src=this.src
    caption.innerHTML=this.alt
  }

  var span=document.getElementById("close");
  span.onclick=function(){
    motai.style.display="none";
  }
</script>
</body>
</html>

这里跟大家补充点知识，方案二里面采用了querySelectorAll这个，以前用的时候，不知道怎么用得更好，今天通过控制台的打印发现这个是一个遍历的，不能拿来就用，道理很简单：

这里有三个图片，所以我们在使用的时候需要指定某个才行。所以这里采用了一个循环，把所有的都遍历出来，其实也就是把方案一中的那个放在循环里面：

realimg.onclick=function(){
  motai.style.display="block"
  moimg.src=this.src
  caption.innerHTML=this.alt
}

放在循环里面的结果：

for (var i=0;i < imgs.length;i++) {
  imgs[i].onclick=function(){
    motai.style.display="block"
    moimg.src=this.src
    caption.innerHTML=this.alt
  }
}

大家注意一下这个：

var imgs = document.querySelectorAll("#container img");

可能大家很自然就想到了jquery，可是为了尽可能的少依赖一些第三方库，笔者这里选择原生的方式。

总结：方案二中要理解querySelectorAll的正确用法，尽量用console.log()来配合调试；第二就是理解这个模态框的原理，是如何巧妙运用的。精华都在demo里面了。

基于HTML5鼠标悬浮出现内容

HTML代码

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <style>
        .content{
            text-align: center;
            height: 50px;
            line-height: 50px;
            width: 100px;
            margin: 100px auto;
            padding: 5px;
            background-color: rgb(196,0,0);
            color: white;
            font-size: 18px;
            position: relative;
        }
        .content:hover{
            cursor: pointer;
        }
        .content:hover:before{
            content: attr(data-content); /*动态获取数据*/
            position: absolute;
            left: 100%;
            width:200px;
            height: 50px;
            font-size: 18px;
            line-height: 50px;
            background-color: rgb(0,196,0);
            margin-left: 12px;
        }
        .content:hover:after{ /*实现小三角*/
            content: "";
            position: absolute;
            left: 100%;
            top: 50%;
            margin: -10px 0 0 -8px;
            width: 0;
            height: 0;
            border-width: 10px;
            border-color: transparent rgb(0,196,0) transparent transparent;
            border-style: solid;
        }
    </style>

    <div data-content="我是一个悬浮提示框" class="content">
        <img src="1.jpg" alt="">
    </div>
</body>
</html>

最后的效果：

这个跟上面的两个不一样的地方就是这个不是点击产生的内容，而是鼠标悬浮的显示内容，这个由于不需要点击事件，所以直接通过css的content: attr()方式，大家可以研究下类似的api用法。

这个方法有点缺陷就是只能显示一个自定义属性的内容，而前面基于JavaScript的点击事件的可以显示多个内容，只要正确获取了属性值就行。
