css3的keyframes如何让动画只执行一次就停下来，分别采用jquery和鼠标悬浮的方式处理

Posted on 2018-05-26 Edited on 2020-10-11

通常，我们在使用css3的动画特效，比如笔者最近就在帮助客户添加一个动画，左右摇摆的动画，动画加上去了，可是鼠标悬浮上去，或者鼠标不悬浮都一直在那里摇摆，看起来不太好，久了眩晕。

@-webkit-keyframes zy{
  10% {
    transform: rotate(15deg);
  }
  20% {
    transform: rotate(-10deg);
  }
  30% {
    transform: rotate(5deg);
  }
  40% {
    transform: rotate(-5deg);
  }
  50%,100% {
    transform: rotate(0deg);
  }
}
@-moz-keyframes zy{
  10% {
    transform: rotate(15deg);
  }
  20% {
    transform: rotate(-10deg);
  }
  30% {
    transform: rotate(5deg);
  }
  40% {
    transform: rotate(-5deg);
  }
  50%,100% {
    transform: rotate(0deg);
  }
}
@-o-keyframes zy{
  10% {
    transform: rotate(15deg);
  }
  20% {
    transform: rotate(-10deg);
  }
  30% {
    transform: rotate(5deg);
  }
  40% {
    transform: rotate(-5deg);
  }
  50%,100% {
    transform: rotate(0deg);
  }
}
@keyframes zy{
  10% {
    transform: rotate(15deg);
  }
  20% {
    transform: rotate(-10deg);
  }
  30% {
    transform: rotate(5deg);
  }
  40% {
    transform: rotate(-5deg);
  }
  50%,100% {
    transform: rotate(0deg);
  }

}

这个是左右摇摆的动画，于是我们确定某个区块(class=”goalelement”)运用它：

.goalelement{
  animation: zy 2.5s .15s linear infinite;
  -moz-animation: zy 2.5s .15s linear infinite; /* Firefox */
  -webkit-animation: zy 2.5s .15s linear infinite; /* Safari and Chrome */
  -o-animation: zy 2.5s .15s linear infinite; /* Opera */
}

这个一放上去就没完没了了，一直摇摆。

于是采用鼠标悬浮：

.goalelement:hover{
  animation: zy 2.5s .15s linear infinite;
  -moz-animation: zy 2.5s .15s linear infinite; /* Firefox */
  -webkit-animation: zy 2.5s .15s linear infinite; /* Safari and Chrome */
  -o-animation: zy 2.5s .15s linear infinite; /* Opera */
}

这样确实不会一直摇摆，可是当鼠标悬浮到这个区域的时候，还是一直摇摆，根本停不下来。

这个时候，我们需要采用一个新的东西（forwards），于是改成如下：

    animation: zy 3.5s .15s linear forwards;
    -moz-animation: zy 3.5s .15s linear forwards; /* Firefox */
    -webkit-animation: zy 3.5s .15s linear forwards; /* Safari and Chrome */
    -o-animation: zy 3.5s .15s linear forwards; /* Opera */

终于停下来了，这个需要给大家科普下：

zy是自己定义的动画帧，3.5s是整个动画的秒数，infinite是永久循环，如果只循环一次则采用：forwards，linear是线性变化

forwards：当动画完成后，保持最后一个属性值（在最后一个关键帧中定义）。

最后还有一个animation-direction 属性（alternate）定义是否应该轮流反向播放动画，默认情况下是normal。具体的可以参考W3C。

好的，这样的话，就算是停下来了。如果想要更加灵活点的话，我们笔者尝试着采用jquery动态添加样式。

<html>
<head>
<script src="http://libs.baidu.com/jquery/1.9.1/jquery.min.js"></script>
<style type="text/css">
    .zy{
        background:red !important;
        height: 200px !important;
        width:200px !important;
    }
</style>
<script type="text/javascript">
$(document).ready(function()
  {
  $(".noctiht-move").mouseover(function(){
    $(this).addClass("zy");
  });    
  $(".noctiht-move").mouseout(function(){
    $(this).removeClass("zy");
  });
});
</script>
</head>
<body>
<div class="noctiht-move" style="background:#98bf21;height:100px;width:100px;margin:6px;">
</body>
</html>

这样就可以把样式添加进去，覆盖掉原来的样式。同样的我们可以把动画写进预先准备好的class中，然后通过鼠标悬浮事件进行动态添加样式，然后根据鼠标离开这个区域额时候，动态的删除样式。
