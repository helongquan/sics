WOW Slider如何去除水印整理

Posted on 2016-07-16 Edited on 2020-10-11

作为前端开发的兄弟们，接触频率较多的无非就那几种，作为现代网站开发中，轮播，旋转，鼠标悬停似乎变得必不可少，这些效果我们可以自己通过js的方法实现或者是css3来实现，当然这是对于比较熟悉代码的同学们而言的，如果对编程一点也不懂的同学估计有点难度；另一种方法，我们可以通过一些生成器或者生成器来完成网站的局部效果，这种方式较为通用，快速而高效，懂得软件的一般操作的从业人员基本就能掌握，废话说了很多，现在我来介绍一款软件：WOW Slider

这是一种幻灯片效果的生成器，操作容易，给我们带来了方便，我们再也不用费脑筋去一个个代码的敲出来了，同时有个美中不足的地方是，这款软件是收费的，而且价格不菲。当然，也有免费版的，但是都会在每一章图片后面自动添加了水印，让人很不舒服，那如何去除这个小小的logo呢？

我开始接触的时候，为了去除这个logo可谓费了不少时间，仍然找不到好的方法，百度搜索的都是如下的一些方案：

1、在wowslider.js里找到show()方法改为hide()，可能有几个show可以一个个试。

这个方法说得很含糊，因为wowslider.js有两个，一个是这个软件模板引擎的wowslider.js，另一个是生成幻灯片之后的在engine1文件夹里面的wowslider.js，这个会让那些初学者分不清楚，后来我根据这个提示两个都改了，貌似没啥用，被我放弃了。

2、在engine文件夹下编辑script.js，Ctrl+F：把var t=”xxxxxxxx”改成 var t=””

截图如下：

这种方式看起来挺详细的，这个是针对V8.2版本的，经过尝试由于版本不一样，没法实现，果断放弃了。而且对比了下这个版本跟最新版V8.7，界面太丑了，还是新版的好看，功能丰富了不少，所以没有使用旧版。

3、找到wowslider.js,打开之后搜索

("contextmenu",function(c){return false}).show()

把show()改成hide()就可以了

这种方法也被尝试了，还是不见好转，看来国内的解决方案有待提升呀，好多都是复制粘贴，转了一圈回来还是没找到。这个解决方案是来自这个网站：http://blog.wangjunfeng.com/archives/475

4、方法四

用firebug 定位 水印 灰色 字体 关键字：right: “5px” 发现在此修改css可将水印去掉

在wowslider.js中 搜索 5px 找到

 ai.append(ae) } K(ai).css({ zIndex: 11, right: "5px", bottom: "2px"}).appendTo(Q);

替换成 

ai.append(ae) } K(ai).css({ zIndex: 11, right: "5px", bottom: "2px", display: "none" }).appendTo(Q);

这种方法具有开创性，避开js,直接通过css的方法去掉这个水印，不过经过尝试，也以失败而告终，后来不知怎滴，我的天才般的想法诞生了。

我的方法是，继承方法四，我在想这个logo不是在右下角嘛，我就直接在那个div里面设置样式，直接，我在软件的模板引擎里面找到wowslider.js，找到

aI("<div>").css({ position: "relative", width: "100%", "font-size": 0, "line-height": 0, "max-height": "100%", overflow: "hidden"})

我之所以能找到这里也是通过开发者工具定位到样式，然后顺藤摸瓜找到这里，然后把上面的待改成

aI("<div>").css({ position: "relative", width: "100%", "font-size": 0, "line-height": 0, "max-height": "100%", overflow: "hidden","margin-bottom":"-30px" })

这样就直接把那个水印给遮住了，效果不受影响，后来测试了一下，发现兼容性有点问题，效果最好的是chrome和safari,有问题的是火狐和IE，那么怎么办呢？

我们还是回到开发这工具，通过设置幻灯片的高度来遮住这个logo即可。虽然这种方法很邪恶但是总比找不到方法又想用的了，再则咱们又不是用于商业用途，如果同学们想用于商业用途，建议还是要去买正版的，用着放心。

针对有问题的怎么办呢？这里我们就需要辛苦下，在生成的WOW Slider文件夹下有个engine1，找到这个文件夹里的style.css打开找到

#wowslider-container1 .ws_images{
    position: relative;
    left:0;
    top:0;
    height:100%;
    max-height:360px;
    max-width: 960px;
    vertical-align: top;
    border:none;
    overflow: hidden;
}

我们就可以对这个高度值进行调整，调小点就可以掩盖掉那个水印，经过自己的测试，大概从360px变成330px即可。

最后针对不同屏幕尺寸编写一个样式即可，高度用一个浏览器尺子插件会更方便，写了这么多，大家可能会觉得这样会比较麻烦。对于这种治标不治本的hack方法不建议同学们学习，最好花点钱来得舒服。

作为补充，如果不觉得麻烦的话，可以把准备好的图片先把高度增加50-70px（高度根据实际情况增加）空白，然后通过WOW Slider把经过高度增加后的图片进行生成，无论是wordpress还是文件夹形式的，然后找到生成后的文件夹里面加过水印的图片部分切掉。同样的，在wordpress插件中先把生成后的压缩包解压，然后剪切掉有水印的部分，然后压缩即可。这样无论是在哪一个浏览器上都不会出现水印，因为图片中根本就没水印了。
