这里说明一下：

这里是调用分类为Personal ESD，它对应的ID是10，五列显示，总共显示10个产品，采用flex_grid.phtml这个模板文件。

type – 这是模块类的标识符，它定义了区块的功能。此属性不应该被修改。

name – 这是名称，其他的区块可以通过此名称引用此区块。
