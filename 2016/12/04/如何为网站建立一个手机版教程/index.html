如何为网站建立一个手机版教程

Posted on 2016-12-04 Edited on 2020-10-11

根据项目需要，为网站弄一个手机版，第一次的时候不知道怎么弄，网上搜了很多讲不清楚，相信很多新手也不知道怎么弄，好吧，我就来献丑了。

首先得确定自己的主机是否支持二级域名，这个直接咨询空间商就行了。

然后，去自己域名解析那里找到添加解析，我这里解析如下：

记录值就是网站域名的IP，跟你的PC版域名一样IP号。

然后去空间里面，新建一个子站点，教程地址：http://www.zhujimiao.com/knowledgebase.php?action=displayarticle&id=17，截图如下：

然后，来到ftp,通过FTP我们在我们的网站根目录，就是www或者是htdocs下的文件中会生成一个m的文件夹，看截图：

打开里面啥也没有，或者是一些提示什么的，都可以把里面的东西删除的，然后我们在m文件夹中存放手机版的网站程序，我们可以随便写点东西作为测试（index.html），然后我们访问我们的手机版网站：这里以www.xxx.com作为我的网站，输入m.xxx.com就可以看到我们的写在m文件夹中的内容了，这样基本算是完成了一个手机版的设置，可是很不智能，就是不会自动识别我们用的是手机打开我们的www.xxx.com还是PC，所以这里我们需要给它做一个跳转，好！

我们继续往下看：

这里我们需要在我们PC版的源码首页中存放一些代码让它做个跳转，我们需要在PC版网站根目录的index文件中添加跳转代码，

这段javascript代码大意思就是匹配设备，如果匹配到iphone/ipad等移动终端的话，将会跳转到指定的地址，这是http://m.xxx.com，这样就让我们的网站变得更智能了。

大功告成！这里的跳转参考了这篇文章的内容，引来参考下：

代码如下：

将以下代码放在首页区

<SCRIPT LANGUAGE="JavaScript">
 function mobile_device_detect(url)
 {
        var thisOS=navigator.platform;
        var os=new Array("iPhone","iPod","iPad","android","Nokia","SymbianOS","Symbian","Windows Phone","Phone","Linux armv71","MAUI","UNTRUSTED/1.0","Windows CE","BlackBerry","IEMobile");
 for(var i=0;i<os.length;i++)
        {
 if(thisOS.match(os[i]))
        {  
  window.location=url;
 }

 }
 //因为相当部分的手机系统不知道信息,这里是做临时性特殊辨认
 if(navigator.platform.indexOf('iPad') != -1)
        {
  window.location=url;
 }
 //做这一部分是因为Android手机的内核也是Linux
 //但是navigator.platform显示信息不尽相同情况繁多,因此从浏览器下手，即用navigator.appVersion信息做判断
  var check = navigator.appVersion;
  if( check.match(/linux/i) )
          {
   //X11是UC浏览器的平台 ，如果有其他特殊浏览器也可以附加上条件
   if(check.match(/mobile/i) || check.match(/X11/i))
                 {
   window.location=url;
   } 
 }
 //类in_array函数
 Array.prototype.in_array = function(e)
 {
  for(i=0;i<this.length;i++)
  {
   if(this[i] == e)
   return true;
  }
  return false;
 }
 }
mobile_device_detect("http://m.xxx.com");
</SCRIPT>

这是第一种方法，倒数第二行中的 http://m.xxx.com 就是要跳转到的手机版网址

方法二，这是用较简单的JS代码，亲测可用：

<script type="text/javascript">
try {
var urlhash = window.location.hash;
if (!urlhash.match("fromapp"))
{
if ((navigator.userAgent.match(/(iPhone|iPod|Android|ios|iPad)/i)))
{
window.location="http://m.xxx.com/";
}
}
}
catch(err)
{
}
</script>

方法三，借用一下作者的第三种方法：

<script type="text/javascript">
// JavaScript Document
function urlredirect() {
    var sUserAgent = navigator.userAgent.toLowerCase(); 
    if ((sUserAgent.match(/(ipod|iphone os|midp|ucweb|android|windows ce|windows mobile)/i))) {
        // PC跳转移动端
        var thisUrl = window.location.href;
        window.location.href = thisUrl.substr(0,thisUrl.lastIndexOf('/')+1)+'m/';

    }
}
urlredirect();
</script>

大功告成！！！
