css实现鼠标悬浮产生阴影的立体效果

Posted on 2018-09-30 Edited on 2020-10-11 In html5

废话少说，直接上demo代码：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
<div class="custoers">
    <div class="custmoerlist">
        <img src="1.jpg"/>
    </div>
</div>
<style>
    .custoers{
        display: flex;
    }
    .custmoerlist{
        padding:15px;
        position: relative;
    }
    .custmoerlist img{
        border:1px dashed #f89a12;
        border-radius: 50%;
        padding:10px;
        transition: 1s ease;
    }
    .custmoerlist:hover img{
        transform: translateY(-10px);
    }
    .custmoerlist:after{
        content: '';
        background: rgba(0, 0, 0, 0.07);
        box-shadow: 1px 0px 20px 2px rgba(0, 0, 0, 0.2);
        border-radius: 50%;
        height: 0px;
        width: 100px;
        z-index: 999;
        bottom: -10px;
        position: absolute;
        left: calc(100%/2 - 50px);
        transition: 1s ease;
        animation-duration:1.5s;
        opacity: 0;
    }
    .custmoerlist:hover:after{
        opacity: 1;
    }
</style>
</body>
</html>
