wordress网站enfold主题头部菜单区域添加语言切换功能

Posted on 2018-05-23 Edited on 2020-10-11

经常看到有客户问，能不能帮我把语言切换功能这个加到wordpress网站的菜单那里，就在菜单旁边就好，放右边，没错….看到这个，可能很多客户会觉得这个不就是很容易的嘛，对于专业人士来说这个不难，但是对于不懂的人来说也不容易。今天就来说说wordpress的头部菜单栏添加语言切换功能的问题。

第一步：定位目标

首先，还是一样的思路，我们通过开发者工具定位到这个是在nav这个标签旁边：

好的，通过多次确认，我们要在这个div的邻居放一个语言切换的功能。

第二步：找到目标对应的源码文件

为了比较精准的搜索到目标，我这里往上走了一级，我把定位定在main_menu这个class上，通过对主题的全源码搜索，我找到了文件：\themes\enfold\includes\helper-main-menu.php，好的，我们来到这个文件所在的目录，打开它：

好的，代码如上，我们将要在这里添加一个select的东西上去。

我们把这个加在$main_nav后面。代码已经准备好了：

<div class="country">
    <select onchange="window.location=options[(this.selectedIndex)].value">
        <option value="/#en">EN</option>
        <option value="/#cn">CN</option>
        <option value="/#fr">FR</option>
    </select>
</div>

这里需要说明一下的是：笔者最开始不是这样加的，最开始的代码是：

<div class="country">
    <select>
        <option value=""><a href="#">EN</a></option>
        <option value=""><a href="#">CN</a></option>
        <option value=""><a href="#">FR</a></option>
    </select>
</div>

可是去前台一调试发现这个根本没法加上链接，后来在一篇文章中，得到了答案，大概是说不能这样加链接的，只能这样加：

<select onchange="window.location=options[(this.selectedIndex)].value">
    <option value="http://www.163.com">232</option>
    <option value="http://www.baidu.com">231</option>
</select>

确定好这个以后，我才改成上面的那种，采用js进行跳转。

好的，最后添加代码后是这样的：

后来客户又变卦了，说这个下拉框那里的样式不是跟他参考的一模一样，后来尝试着用css改变这个样式，可是用css改变option的样式貌似都没啥用，这个时候找到一个方法，放弃采用select的方法去解决这个问题。转用jquery的方式。

方法如下：

第一步：在header.php文件中引入：

<script src="https://cdn.bootcss.com/jquery/3.3.1/jquery.min.js"></script>
<script type="text/javascript">
    $(document).ready(function(){
            $(".select_box").click(function(event){   
                event.stopPropagation();
                $(this).find(".option").toggle();
                $(this).parent().siblings().find(".option").hide();
            });
            $(document).click(function(event){
                var eo=$(event.target);
                if($(".select_box").is(":visible") && eo.attr("class")!="option" && !eo.parent(".option").length)
                $('.option').hide();                                      
            });
            /*赋值给文本框*/
            $(".option a").click(function(){
                var value=$(this).text();
                $(this).parent().siblings(".select_txt").text(value);
                $("#select_value").val(value)
             })
        })
</script>

意思就是：引入jquery库，然后运用select插件。好的完成引入第三方库后，我们来到这个网站头部，好的控制语言切换的文件，还是跟前面那样：\themes\enfold\includes\helper-main-menu.php，打开文件。
