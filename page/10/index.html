先是获取这个元素的自定义属性：data-src的值，也就是图片地址，那后把这个传值到当前dom的背景图片属性，然后设置为100%显示。然后就是如何选择当前dom的子元素的呢？这里有个jquery方法children()，顺便科普一下关于获取当前子元素的方法：

Jquery获取子元素的方法有2种，分别是children()方法和find()方法。

1.children()方法：获取该元素下的直接子集元素

2.find()方法：获取该元素下的所有子集元素

3.children()方法获取ul下面直接子集元素li：$(“ul”).children(“li”)

4.find()方法获取ul下所有元素li：$(“ul”).find(“li”)

5.children和find的区别：children只会查找直接子集，而find会跨越层级查找，一直找到没有为止

好的，我们言归正传，这里我们让当前元素的子集span元素隐藏：

$(this).children('span').css({'display':'none'});

这样就完成了一个鼠标悬浮的过程，接着就要实现鼠标离开这个dom元素发生的事，顺便说下，笔者在处理这个的时候，总是有问题，问题就在于，鼠标悬浮上去是达到了背景图片显示了，可以是鼠标离开后呢，背景图片依然在，就是不恢复到最初的样子，后来进过查阅资料发现这个是因为，我们没有指定离开后的背景图片。笔者这里单单只是采用了：

$(this).removeAttr('background-image');
$(this).removeAttr('background-size');

甚至，笔者这样写（笔者企图通过这种方式来删除这个属性）：

$(this).css({'background-image':'none'});

这样都是不行的，不用用这种方式来解决这个问题。

解决方案就是必须要重新指定一个背景给当前dom，以替换才行，如下：

var currentBackgroundColor=$(this).css("background-color");
$(this).css({'background':currentBackgroundColor});
$(this).children('span').css({'display':'inline-block'});

代码解释：

由于笔者的这个列表的背景颜色比较复杂，所以这里不能只替换一种颜色，

所以，这里选择当前的背景颜色，然后把这个属性值保存在一个变量(currentBackgroundColor)中。然后把这个变量传递到鼠标离开后的元素背景属性里面去，最后再把悬浮的时候的文字显示出来。

这样就以一个背景的方式来展示图片了，用这个方案的好处就是我们能够让这个背景图片显示在我们的dom里面。

获取自定义的属性，动态生成图片插入dom中

除了用前面的那个方法之外，这里我们还可以用这种方法处理鼠标悬浮的图片显示，鼠标离开图片消失。

HTML代码不变，css代码我们在原来的基础上添加一条：

.rrg img{
    max-width: 150px;
    max-height: 150px;
}

这个就是用来约束动态生成的图片的大小，而这个图片的大小是要跟这个盒子大小一样的尺寸，以便不把这个盒子撑变形了。好的，我们来看下js：

<script>
    $(document).ready(function(){
        $(".rrg").hover(function() {
            var newbackground = $(this).attr("data-src");
            //console.log(newbackground);
            $(this).append("<img src="+newbackground+">");
            $(this).children('img').css({'transform':'rotate(-45deg)'});
            $(this).children('span').css({'display':'none'});
        }, function() {
            $(this).children('img').remove();
            $(this).children('span').css({'display':'inline-block'});
        });
    })
</script>

这个意思已经很明了了，获取自定义属性值，然后动态添加一个img标签，然后把属性值插入到img标签中，然后进行选择回正，鼠标离开的时候就删除掉这个img标签，然后把里面的文字显示出来。

这就是两种方法：

  1. 方法一其实是最好的，不过有个小小的缺点就是背景图片没法旋转
  2. 方法二就是相对来说对图片的尺寸要求约束多了点，图片的尺寸必须是跟盒子的尺寸一致，对图片的css处理相对来说灵活点

如果能够解决背景图片的旋转，那么建议采用方案一。

Ajax点击不断加载数据列表,实现加载更多的功能

Posted on 2018-10-14 Edited on 2020-10-11 In javascript

我们来说说ajax逐个加载数据，类似这样的数据列表是怎么实现的，

先是显示10个，然后点击加载更多，再显示10个·····

一、思路

一般使用ajax加载的话，一下会加载全部数据，这次要控制量的加载，那就要用到判断，判断加载到10个停止加载，后面通过点击按钮，在接着加载完的10个后面继续加载。

要想控制只加载10个怎么办呢。不能通过遍历的 i 来判断 10个，因为加载完10个后后面还要加载，这样下一个10个就不好判断了，所以需要定义一个新的变量，来计算加载的个数,

可以这样写：

var ci = 0;
for(var i = 0; i < data.list.length; i++){
　　ci++; 
　　if(ci> 10){
　　break;
　　}
}

然后需要再加载10个，再调用这个方法，所以这个方法需要声明一个函数名，下次需要用的时候在调用，如果需要传参也可以。现在还有个问题，第一次进来加载完10个后，需要再加载前10个数据接着后面的json数据，怎么办呢？

没关系，可以通过调用上面定义好的函数，然后传参。参数怎么计算呢？

先想一个参数和什么有关系，和 i 有关系，i 和什么有关系呢？或者说什么可以影响 i 呢？

好像只有它的值才会影响（那不是废话吗），这样的话它的值就不能是一个不变的数了，只能是一个变量，那么变量从哪里来呢？

别忘了我们还有一个点击事件，先定义一个 点击次数的变量 var clickNum = 0，因为每次加载时10个，所以 i 的值应该为：

i = 10*clickNum，这是每次加载第一个数据的索引值。这样我们就解决了上面那个问题。

此时还有问题要解决，当数据全部点击加载完后，点击按钮需要隐藏，那怎么计算数据加载完了呢？

我们可以通过 点击的次数clickNum 来计算，因为每次加载10个，所以可以计算出一共需要加载的次数 parseInt((data.list.length)/10)+1，为什么加载的次数要加1呢？

因为parseInt()是取整，比如21/10=2，但实际需要加载 3 次，所以要加 1 ，巧的是我们第一次加载不用点击，浏览器已加载进来就读取了10个数据，所以clickNum = parseInt((data.list.length)/10)，当clickNum == parseInt((data.list.length)/10) 时隐藏点击按钮。

思路基本清晰了

利用jquery和css巧妙地把鼠标悬浮头部菜单的时候logo图片替换以及头部固定的时候变成另外一种颜色

Posted on 2018-10-08 Edited on 2020-10-11 In javascript

最近有个客户要求非常特殊，非要把网站做得跟大疆官网一样，其中就有一个鼠标悬浮在头部菜单栏的时候，logo的图片会切换，由于logo不是字体图标，而是图片，因此没法采用css的hover了。那么怎么办呢？且看笔者的方法。

因为客户的网站最开始是玻璃效果的，这个时候logo需要是白色的，因此不得不做一张白色的logo，

当然菜单的文字也是白色的。

接着呢，当鼠标悬浮到菜单栏和logo所在的区域的时候，背景颜色由透明状变成白色，logo图片由白色替换成黑色的，菜单文字变成黑色。

因为现在是在最顶部，当我们鼠标往下滚的时候，头部菜单栏是固定在顶部的，这个时候，持续保持这状态或者logo颜色变成另外一种，

面对一系列的问题，笔者的解决方案是结合开发者工具找到目标元素，

当我们往下滚动的时候，会发现：

这里有个明显的差异就是没有了av_header_transparency。

跟这个相关的，笔者都圈出来了，分别是：id为header_main的头部区块，class为av_header_transparency的玻璃特效的类，接着就是目标元素img。这个是是enfold主题下的，其他主题可能不一样，这里仅仅是用作参考。笔者采用jquery的方式对鼠标悬浮动态的插入图片地址：

<script src="<?php bloginfo('template_url'); ?>/js/jquery-3.1.1.min.js"></script>
<script>
var jQuery_3_3_1 = $.noConflict(true);
jQuery_3_3_1(document).ready(function(){
    //如果
    if(jQuery_3_3_1("header").is(".av_header_transparency")){
        jQuery_3_3_1("#header_main").hover(function(){
            jQuery_3_3_1("#header_main .logo img").attr("src", '/wp-content/uploads/2018/10/logo-balck.png');
           jQuery_3_3_1("#header_main").css({'background':'#fff !important'});
         },function(){
            jQuery_3_3_1("#header_main").css({'background':'transparent !important'});
            jQuery_3_3_1("#header_main .logo img").attr("src", '/wp-content/uploads/2018/10/logo-white.png');
        });
    }
    jQuery_3_3_1("#header_main").mouseenter(function () {
        jQuery_3_3_1("#header_main").css({'background':'#fff !important'});
          jQuery_3_3_1("#header_main .logo img").attr("src", '/wp-content/uploads/2018/10/logo-balck.png');
    });
    jQuery_3_3_1("#header_main").mouseout(function () {
          jQuery_3_3_1("#header_main .logo img").attr("src", '/wp-content/uploads/2018/10/logo-white.png');
    });

})
</script>

这个代码目的就是想要实现一个鼠标悬浮的时候，logo变成黑色，背景变成白色，离开的时候，logo变成白色，背景回到最初的样子（透明色），这个是在头部是透明状态下的情况发生的，

jQuery_3_3_1("header").is(".av_header_transparency")

如果没有这个class（.av_header_transparency）的情况下，就用mouseenter和mouseout这两个进行控制logo的切换，经过实践笔者发现总是不尽如人意，虽然达到了部分的预期。就是当笔者解决了在透明状态下的鼠标悬浮是黑色的，离开是白色的，那么滚动的时候logo是白色的，要么是解决了最初透明状态的白色logo，鼠标悬浮式黑色的，滚动后的logo还是白色，可是当滚动到顶部的透明状态的鼠标悬浮的时候，logo还是黑色的。所以不理想。那么笔者就在想，先解决不是透明状态的情况，就是鼠标滚动后固定在顶部的菜单栏，这个时候颜色和背景完全可以采用默认的或者是jquery的方式搞定，然后呢当来到顶部透明状态的时候，只需要写一个鼠标悬浮的css样式即可。只是这个地方需要把logo的图片隐藏，然后把logo当做背景的方式添加到那个地方即可。事实证明我的猜想是对的。看下面。

方法如下：

.av_header_transparency:hover .logo a {
    background-image: url('/wp-content/uploads/2018/10/logo-balck.png');
    height: 65px;
    width: 100%;
    display: inline-block;
    background-repeat: no-repeat;
    background-size: 100%;
}
.av_header_transparency:hover .logo a img{
  visibility: hidden;
}

两条代码搞定，意思给大家解读一下：

当鼠标悬浮到有透明状态的头部菜单的时候，把logo所在的img那里隐藏，而不是display:none，依旧占据这个位置。然后在这个位置的上一层标签那里，也就是a标签，那里添加一个背景图片，为了图片能够完整的显示，这里的background-size采用100%。其他样式的大家可以根据情况自己设置了。

后来解决这个问题，笔者仅仅是采用了后面的css就搞定了，这个值得笔者反思，也许很多的时候，样式问题可以用css搞定，可能有的时候，css是没法搞定的，需要js来弄，可是很多时候我们不能只是单纯的只用一种方式来解决问题，可以尝试利用多种方式各自的优点和特性来解决问题。我相信这才是写这篇文章最宝贵的地方。

利用jquery的方式对wordpress网站的菜单嵌入文字或者内容

Posted on 2018-09-28 Edited on 2020-10-11 In wordpress

很多时候，我们在做wordpress网站的时候，特别是自适应的时候，手机版会折叠成三条横线，或者是woocommerce的网站，会有个购物车的标志，客户很喜欢在这个标志旁边添加一些内容，比如购物车文字，菜单的文字，比如menu。根据以往的经历我们通常可以通过主题的源码找到相应的位置进行添加，不过，如果不是一个小功能，或者需要涉及到后端的php代码的时候，我们尽量不要修改源码，以确保网站的安全和结构的稳定。

相对合理的方式是通过js的方式动态嵌入进入，这样我们就不需要添加修改源码了。做法如下：

需求一

想要在woocommerce购物车旁边添加文字类似的需求：

这个时候，我们需要在网站的后台那里，进入到公共底部的文件：footer.php，当然header.php也可以，在</body>前面添加如下代码：

// 引入jquery
<script src="https://cdn.bootcss.com/jquery/3.3.1/jquery.min.js"></script>
<script type="text/javascript">
    var jQuery_3_3_1 = $.noConflict(true);  //处理jquery重复进入的冲突
    var cart = jQuery_3_3_1("<span class='geggh carts'>cart</span>");   //添加动态内容
       jQuery_3_3_1(".cart_dropdown_link span").append(cart);   //插入创建的动态内容到目标位置
</script>

经过css优化，我们就可以把这个变成上面截图的那样。

需求二

想要在菜单位置添加内容类似的需求：

红色框住的地方是没有文字的，客户想要在这里添加文字，以便让用户知道这个是折叠起来的菜单。

还是跟前面一样的：

// 引入jquery
<script src="https://cdn.bootcss.com/jquery/3.3.1/jquery.min.js"></script>

    var menus = jQuery_3_3_1("<span class='geggh'>menu</span>");   //添加动态内容
       jQuery_3_3_1(".av-small-burger-icon>a").append(menus);   //插入创建的动态内容到目标位置，这个需要通过开发者工具定位真正的目标位置
</script>
