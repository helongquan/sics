获取Magento购物车信息和当前网站店铺的信息，以及获取用户登录状态及登录用户信息

Posted on 2018-05-25 Edited on 2020-10-11

Magento 获取购物车信息常用代码

我们在做magento模板过程中，常常希望得到购物车的金额，货币信息等。

获取magento网店当前货币code：

echo $currency_code = Mage::app()->getStore()->getCurrentCurrencyCode();

获取magento网店当前货币符号信息：

echo Mage::app()->getLocale()->currency(Mage::app()->getStore()->getCurrentCurrencyCode())->getSymbol();

得到当前购物车中的全部产品的金额：

$grandTotal = Mage::getModel('checkout/cart')->getQuote()->getGrandTotal();

这样货币符号加上金额我们就得到了具体的当前金额了

得到当前购物的所在货币方式的全部金额

getQuote()->getGrandTotal();
getLocale()->currency(Mage::app()->getStore()->getCurrentCurrencyCode())->getSymbol().$grandTotal

在Magento购物车应用开发中，也经常需要判断当前用户的登录状态，并获取当前登录用户的相关信息。

首先，我们需要判断当前用户的登录状态，通过如下方式。如果用户已登录，则返回真，否则返回假。

Mage::getSingleton('customer/session')->isLoggedIn();

当判断用户已经登录的情况下，即可获取当前登录用户的相关信息。如用户名，姓名，邮件等。

$customer = Mage::getSingleton('customer/session')->getCustomer();
echo $customer->getName();
echo $customer->getEmail();

这里不仅可以通过getAttribute()的方式获取系统内置属性，如果你添加了一个customer的自定义属性，也可以通过此种方式获取该值。

$customer = Mage::getSingleton('customer/session')->getCustomer();
echo $customer->getFacebookUid();

店铺的信息获取方式

Magento的多店铺功能非常强大，通过多店铺功能，我们可以在一个Magento应用当中创建在外观上看来完全不相关联的多个店铺，并且可以绑定不同的域名。在Magento的开发当中，经常需要获取当前所在店铺的一些相关信息，例如店铺id，店铺名称及当前店铺所属Website的ID值。通过Magento的Mage类，我们可以获取到关于店铺的所有信息。

//获取店铺对象
<?php echo Mage::app()->getStore();?>

//获取当前店铺ID
<?php echo Mage::app()->getStore()->getStoreId();?>

//获取当前店铺Code，该Code在创建店铺时填写
<?php echo Mage::app()->getStore()->getCode();?>

//获取当前店铺所属的Website ID
<?php echo Mage::app()->getStore()->getWebsiteId();?>

//获取当前店铺的Name
<?php echo Mage::app()->getStore()->getName();?>

//获取当前店铺的状态
<?php echo Mage::app()->getStore()->getIsActive();?>

//获取当前店铺的URL
<?php echo Mage::app()->getStore()->getHomeUrl();?>

这样就得到当前购物车中的全部金额了！
