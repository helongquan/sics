分享几款优秀的web前端框架

Posted on 2016-06-04 Edited on 2020-10-11

从事web前端已经有一些时间了，解决过不少棘手的问题，现在回头想想曾经的麻烦都是因为缺少充分的准备，试想一下，如果当初能够准备好充分的UI库，充分的资源，能够多点时间去了解一些优秀的UI库是不是能够面临新的问题的时候能够快速地给出很好的解决方案，记得曾经就遇到过一个问题，结果客户那边催促得紧，这边还没找到一个良好的解决方案，这种心情就像热锅上的蚂蚁。

中间省略部分废话…

首先要推荐的就是的弹窗UI框架layerUI，用过的朋友会觉得非常省时间，尤其是在一些修改网站的局部效果的时候会用得上，与其各个地去添加js代码，还不如直接引用这个框架来得省事，当然这里提醒一下，对于初学者不建议过于依赖这些UI框架。

参考网址：http://layer.layui.com/

下载地址：http://pan.baidu.com/s/1eSuLke2 密码：pp0z

引用官网的话说：

layer是一款近年来备受青睐的web弹层组件，她具备全方位的解决方案，致力于服务各水平段的开发人员，您的页面会轻松地拥有丰富友好的操作体验。
在与同类组件的比较中，layer总是能轻易获胜。她尽可能地在以更少的代码展现更强健的功能，且格外注重性能的提升、易用和实用性，正因如此，越来越多的开发者将媚眼投上了layer（已被2772810人次关注）。layer兼容了包括IE6在内的所有主流浏览器。 她数量可观的接口，使得您可以自定义太多您需要的风格，每一种弹层模式各具特色，广受欢迎。当然，这种“王婆卖瓜”的陈述听起来总是有点难受，因此你需要进一步了解她是否真的如你所愿。

当我们在官网下载他们的demo后你就会发现，其实引用这个是相当方便的，头部需要的引用的文件如下：

<script type="text/javascript" src="js/jquery-2.2.3.min.js"></script>
<script src="layer/layer.js"></script>

中间是一些HTML代码，当然其中也可以添加一些js代码，这里我以图片相册功能为例，添加三个ID中的图片相册功能，代码如下：

<script>
    +!function(){
        layer.ready(function(){
            layer.photos({
                photos:'#LayerPhoto'
            });
        });
    }()
</script>

<script>
    +!function(){
        layer.ready(function(){
            layer.photos({
                photos:'#zb-tupian'
            });
        });
    }()
</script>

<script>
    +!function(){
        layer.ready(function(){
            layer.photos({
                photos:'#zb-zhaopian'
            });
        });
    }()
</script>

完整的demo下载地址：http://pan.baidu.com/s/1pL9vUCb 密码：fivl

中间的一些写法大家可以直接参考官网的文档，针对不同的弹窗效果遵守不同的写法。

第二个值得推荐的web前端框架是：jQuery WeUI

相对于bootstrap而言，这个框架算是比较轻量级的了，他们的功能大同小异，随着UI框架的使用，遵循的原则大致相同，即先去官网下载这个库，把下载的这个库根据官网的使用说明对这个库的引用，然后根据使用规则也就是所谓的API，把他们的API用在HTML文件的各个角落，然后在浏览器上打开这个html文件，刷新一下就可以看到奇迹发生了，用了一遍之后，你会发现你快爱上它了…

好吧，废话少说，把这个库的官网给贴上：http://lihongxun945.github.io/jquery-weui/

jQuery WeUI的下载地址：http://pan.baidu.com/s/1jI4g2lK 密码：wlwi

这里再把这个库的demo地址贴上：http://lihongxun945.github.io/jquery-weui/dist/demos/

先去体验一把，看看是不是自己想要的那种效果，然后再使用也不迟，嘻嘻~
